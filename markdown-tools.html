How-To Add Your Tool

  • Fork the /markdown-tools repo on GitHub
  • Add a new entry in the tools.yml file and fill out all fields. Example: 
    - title:    pandoc
  author:   John MacFarlane et al
  formats:  [html,latex]
  github:   jgm/pandoc
  web:      http://pandoc.org
  language: haskell
  • Thanks!
