How-To Add Your Tool

  • Add a new entry in the tools.yml file and fill out all fields. Example:

  formats:   [html (+epub), latex (pdf)]
  templates: Custom (Pandoc)
  github:    jgm/pandoc
  language:  Haskell
  web:       http://pandoc.org
